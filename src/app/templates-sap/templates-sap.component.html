Templates de SAP

Archivo Descripcion

1. ESTRUCTURAS_IE_SPIRIT_V7

Template para estructuras IE:

Este es el template principal para solicitar la parametría de nuevos números de parte en un cambio de ingeniería. En él se indican todas las especificaciones y peculiaridades a considerar de cada proyecto, para la transferencia y creación de dichos números en nuestro sistema NP1.

2. NEW_COMP_ALLOC

Template para Component Allocation:

Template utilizado para solicitar la asignación de materiales (alocación) de acuerdo a la operación/proceso en la que son consumidos o utilizados dentro de la celda de producción, básandonos en el flujo que tiene nuestra ruta.

3. ZMASSCA01_Rutas_Template_V2

Template para rutas:

Este template es para solicitar cualquier tipo de modificación que se necesite hacer en la ruta productiva o ruta de budget en nuestro periodo de planeación financiera.

4. ZMASSCS01_BOM_TEMPLATE

Template para BOMs:

Este template es utilizado para solicitar cualquier modificación que se necesite hacer en nuestra BOM productiva, ya sea cambio de materiales, cantidad o modificación en los designadores (BOMs electrónicas). Cualquier cambio solicitado mediante este template debe estar sustentado por la modificación ya realizada en BOM de R&D de lo contrario la solicitud/ticket debe de ir acompañado por una desviación.

5. ZMASSCR01_WC_TEMPLATE

Template para Workcenters:

Template utilizado para solicitar la creación de nuevos work Centers ya sea de equipos/procesos o de nuevos recursos (PSS).

6. STOCK_PLACEMENT_PSA_TEMPLATE

Template para Stock Placement:

Template que nos ayuda a solicitar la modificación de los parámetros de Stock Placement y de Product Supply Area.

7. TIME_MANAGEMENT_TEMPLATE

Template para time management:

Template que nos ayuda a la realización de time management: el balanceo de nuestra línea de producción y la toma de tiempos.

8.CLASIFICACION_ARANCELARIA_TEMPLAT

Template para clasificacion arancelaria:

Archivo utilizado para la solicitud de parámetros al área de tráfico para nuevos números de parte (cambios de ingeniería).
