IE Maestro

Descarga automatica de formatos recurrentes de IE:

Este archivo te servira para llenar de manera automatica los formatos de alta de estructuras, component allocation y rutas.

Recomendiaciones de uso: Para ejecutar es necesario tener abierta la aplicacion de SAP, ya que la informacion es obtenida directamente de esta aplicacion.

BOMs con referencias

Descarga automatica de BOMs de PCBs con referencias:

Este archivo te servira para descargar la BOM de una ilimitada cantidad de numeros de parte, usualmente utilizada para enviar informacion a ingenieros de placement.

Recomendiaciones de uso: Para ejecutar es necesario tener abierta la aplicacion de SAP, colocar los numeros de parte de las PCBs con terminacion 00 y una vez finalizada podras revisar el file en la carpeta de descargas.

Consulta Masiva de tablas en SAP

Descarga automatica de informacion de tablas en SAP:

Este archivo te servira para descargar informacion de tablas como MARA, MARC, MLGN, MKAL, MARD, MBEW, MAST, MAPL, PGMI, STKO, LQUA, LTAP, junto con una breve descripcion del uso de cada tabla.

Recomendiaciones de uso: Para ejecutar es necesario tener abierta la aplicacion de SAP, ya que la informacion es obtenida directamente de esta aplicacion.
